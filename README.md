# Password Strength Checker

A simple Python program that checks the strength of a password based on common security rules.

## Features

* Checks password length
* Detects numbers in the password
* Detects uppercase letters
* Detects special characters (`!@#$*_`)
* Prevents passwords with spaces
* Provides suggestions to improve weak passwords

## Requirements

* Python 3.x

## Password Strength Levels

* 0 Points → Very Weak
* 1 Point → Weak
* 2 Points → Medium
* 3 Points → Strong
* 4 Points → Very Strong
